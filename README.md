# PopX React Task

A responsive React JS application built as part of the PopX Frontend Developer qualifier task.

## Features

* Welcome Screen
* Login Page
* Registration Page
* Account Settings Page
* React Router Navigation
* Responsive UI
* Clean Component Structure
* Built with React + Vite

## Tech Stack

* React JS
* React Router DOM
* Vite
* CSS3

## Installation

Clone the repository:

```bash
git clone https://github.com/birralaxman36/popx-react-task.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
